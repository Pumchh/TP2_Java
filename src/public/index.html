Présentation

Polytech Dijon est une école d'ingénieur publique de l'Université de Bourgogne créee en 1991. Les frais d'inscription s'élèvent à 601 euros (gratuits pour les boursiers). Elle bénéficie de toutes les insfrastructures de l'UB : installations sprotives, bibliothèque, résidences universitaires, bibliothèque etc.
